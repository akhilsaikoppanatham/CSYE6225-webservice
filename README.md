# Assignment-4

Pre-requisites:
- Install Node.js 

Getting Started:
- Clone the repository
    - git clone https://github.com/Amal-Mat/webservice.git

- Install Dependencies
    - cd webservice
    - npm install


- Build and run the project
    - npm start

Testing:
- The test case is written using Mocha and Supertest -

    - "mocha": "^9.2.0",
    - "supertest": "^6.2.2"

- Running tests using NPM Scripts
    - npm run test
